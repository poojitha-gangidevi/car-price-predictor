 🚗 Car Price Predictor

A Machine Learning web application that predicts the resale price of used cars based on company, model, year, fuel type, and kilometers driven.

🔹 Technologies Used

* Python
* Flask
* Scikit-learn
* Pandas
* NumPy
* HTML
* CSS
* Bootstrap
  
 🔹 Features
* Dynamic company and model selection
* Real-time car price prediction
* Responsive user interface
* Machine Learning based prediction model
